# Cisco-Packet-Tracer-
Connect Devices Using Wireless Technologies

I completed a Cisco Packet Tracer lab where I connected multiple end devices using different wireless technologies in Physical Mode only. First, I replaced a laptop’s Ethernet module with a wireless NIC, joined it to the “Employee” SSID using the pre‑shared key “Cisco123,” obtained a DHCP IP address, and verified connectivity by browsing to the internal web server at office.srv. Next, I enabled Bluetooth on an office tablet and a Bluetooth speaker, paired them, and tested audio playback using the tablet’s music player to confirm the short‑range wireless link. Finally, I enabled Bluetooth and cellular tethering on a smartphone, paired it with a user laptop, tethered the connection, confirmed that the laptop received an IP address from the cellular network, and verified internet access again by reaching office.srv through the phone’s 3G/4G link.
